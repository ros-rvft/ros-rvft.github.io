# News

All updates to the ROS-RVFT guidelines.

* **March 2026** — New exemplar added: UPPAAL automated verification toolchain for ROS 2 timing and scheduling analysis ([PE2](/guidelines/guideline-pe2)).
* **March 2026** — New exemplars added: R2D2 and ROFER ROS 2 fuzzers for noise injection and robustness testing ([MTA1](/guidelines/guideline-mta1)).
* **March 2026** — New exemplars added: FRET for structured natural language specification ([SDB1](/guidelines/guideline-sdb1)) and the FRET+Ogma+Copilot automated monitor generation pipeline ([MTA2](/guidelines/guideline-mta2)).
* **March 2026** — New contribution templates: you can now suggest tools, report gaps, or request updates directly via [GitHub Issues](https://github.com/ros-rvft/ros-rvft.github.io/issues/new/choose) — no Git required.
* **March 2026** — Website refresh: updated contact info, copyright, and contribution guidelines.
* **March 2026** — Site migrated to Jekyll with Minimal Mistakes theme. New sidebar navigation, search, and table of contents on guideline pages.
* **July 2024** — Initial release of the guideline catalog accompanying the TSE paper.
